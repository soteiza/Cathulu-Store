Bienvenido a Cathulu Store

Mi propuesta de proyecto trata sobre una aplicación web la cual sirve para gestionar una tienda y su personal. En este caso una tienda de juegos de mesa, la cual cuenta con las múltiples competencias enseñadas en el curso, además de algunas que fueron agregadas por mi cuenta.

La página presenta un menú lateral el cual muestra todas las propuestas que idee para la aplicación web. Entre estas funcionalidades se encuentran: Gestión del personal de la tienda, ver los juegos de mesa que vende la tienda, redirección a página en caso de encontrarse con una página inexistente.

Para realizar lo anteriormente mencionado, utilicé dos API's. La primera API es usada para poder obtener los juegos de mesa de la página. Esta fué sacada desde la página Boardgameatlas. La segunda API es la proporcionada por usted en clases, la cual se utiliza para manejar al personal de la tienda.
